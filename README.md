# Westongame React Boilerplate

## How to start ?
Before launching the project run
- yarn

Then, during development use
- yarn run dev

## How to lint ?
- yarn run lint

## How to test ?
- yarn run test

## How to build production version ?
- yarn run build


## Dev Dependencies:
- Webpack
- ES6
- ESLint (Modified Airbnb)
- ShortId
- Jest + Enzyme
- CSSModules, Stylus, Autoprefixer

## Dependencies:
- React
- React Router
- Redux + ReduxThunk
- ClassNames
- Lodash
- Axios
